Deployment Guidelines:
  1. Click the download button above to save your completed .md file.
  2. Place this file inside your server's posts/ directory folder.
  3. Open your master index catalog at posts/index.json and prepend your new file name directly into the array structure list: 
    {
  "posts": [
    "your-new-filename.md",
    "welcome-to-my-blog.md"
  ]
}
